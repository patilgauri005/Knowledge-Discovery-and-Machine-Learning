# KDML Lab 1 – Exploratory Data Analysis (EDA)

## Objective
To perform exploratory data analysis on the dataset to understand its structure, statistical properties, correlations, and detect outliers.

## Files in This Folder
- KDML_Pract1.ipynb → Jupyter Notebook implementation
- background_noise_focus_dataset.csv → Dataset used

## Steps Performed
1. Data Loading
2. Data Inspection
3. Statistical Summary
4. Missing Value Analysis
5. Univariate Analysis
6. Correlation Heatmap
7. Outlier Detection using IQR

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
